# 📚 Flashcard Quiz App (CodeAlpha Internship Task 1)

## 📌 Project Description
This is a simple Flashcard Quiz App built using Streamlit.  
It helps users study using interactive flashcards with questions and answers.  
Users can navigate between cards, show answers, add new flashcards, and delete existing ones.

---

## ✨ Features
- 📖 View flashcards (Question → Answer format)
- 👁 Show/Hide answer button
- ⬅ Next and Previous navigation
- ➕ Add new flashcards
- 🗑 Delete flashcards
- 🧠 Simple and clean UI

---

## 🛠 Tech Stack
- Python
- Streamlit
